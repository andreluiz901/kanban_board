# 📝 Kanban Board - Backend

Bem-vindo ao **Backend do Kanban Board**! Este projeto é a base de um quadro kanban onde você pode gerenciar quadros(boards), colunas(columns) e cartões(cards) de maneira eficiente. Totalmente integrado com a ideia de **Drag and Drop** no frontend, o backend lida com a lógica de negócios complexa para garantir uma experiência fluida e responsiva.

## 🚀 Descrição do Projeto

Este é um backend completo para um quadro kanban, no qual é possível:

- Manipular quadros, colunas e cartões.
- Adaptar as respostas para um frontend com rotas para ordenar a partir de um drag and drop.
  
O projeto foi desenvolvido como forma de estudo, focando em soluções robustas para lidar com entidades interdependentes que precisam ser atualizadas de forma rápida conforme o movimento do usuário. Neste trabalho busco demonstrar habilidades com desenvolvimento backend, validações e resolver desafios complexos de negócio.

Aqui estão as principais tecnologias utilizadas e seus propósitos no projeto:

- **Node.js**: Runtime JavaScript utilizado para o desenvolvimento do servidor backend.
- **NestJS**: Framework utilizado para estruturar e organizar o projeto em módulos e serviços de maneira eficiente e escalável.
- **Prisma**: ORM utilizado para interagir com o banco de dados de forma segura e otimizada.
- **class-validator**: Ferramenta para garantir validações robustas em requisições de entrada.
- **JWT (JSON Web Token)**: Usado para autenticação e autorização seguras, garantindo que somente usuários autenticados possam acessar determinadas rotas.
- **Swagger**: Documentação automática de APIs, facilitando o entendimento e o uso dos endpoints disponíveis no sistema.

## 🌟 Features

- Criação de conta de usuário com autenticação via email e senha.
- Login seguro e geração de token JWT.
- Criação de quadros kanban personalizados por usuário.
- Criação e manipulação de ordem de colunas dentro dos quadros.
- Criação e movimentação de ordem de cartões dentro das colunas.
- Movimentação de colunas e cartões com ajuste dinâmico da ordem entre colunas e dentro da própria coluna.
